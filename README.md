# Property Recommendation System

The Property Recommendation System is a web-based application designed to help users discover and explore suitable properties based on their preferences and needs.

The system provides a user-friendly interface where users can browse featured properties, view detailed information, and receive ranked recommendations. Each property is evaluated using a scoring mechanism to highlight the most relevant and high-quality options.

## ✨ Features

* 🏠 Browse featured and recommended properties
* 📊 Property ranking system based on scores
* 👤 User profile managmetn
* 🔍 View detailed property information
* 🔐 User authentication (login/logout system)

## 🛠️ Technologies Used

* Frontend: HTML, CSS, Java Script
* Backend: PHP
* Server: Apache (XAMPP)
* Database: JSON

## 🎯 Purpose

This project aims to simplify the process of finding properties by providing intelligent recommendations and an organized browsing experience.
